# 💾 데스크탑 PC 하드웨어 분석 및 표준 사양 제안 프로젝트

## 1. 프로젝트 개요 및 데이터 출처

본 프로젝트는 대규모 웹 데이터를 기반으로 CPU 및 GPU 성능 변화를 분석하고, 2025년 게이밍 PC 권장 사양을 도출합니다. 모든 분석은 공개된 데이터를 기반으로 수행되었습니다.

### 1.1. 데이터 출처

| 구분 | 데이터 종류 | 사용 목적 |
| :--- | :--- | :--- |
| **Wikipedia** | 하드웨어 제원 정보 (모델명, 출시일) | 분석 대상 하드웨어 목록 및 시계열 기준 확보 |
| **PassMark** | 벤치마크 성능 점수 (CPU/GPU) | 하드웨어 성능의 정량적 지표 확보 |
| **PCGamingWiki** | 게임 요구 사양 목록 | 성능 분석의 최종 목표 설정 및 사양 도출 |

## 2. 필수 안내 사항

### 2.1. 파일 및 실행

* **메인 실행 파일:** 프로젝트의 모든 데이터 수집, 정제, 분석 과정은 `BigData.ipynb` 파일 내에 구현되어 있습니다. (Jupyter 또는 Colab 환경 권장)
* **HTMLS 폴더:** 프로젝트의 데이터 수집 과정 중 **PassMark**에서 벤치마크 점수를 가지고 오는 기능을 실행했을때 **데이터가 1~3개를 불러오거나** 또는 **403 error**가 발생할 경우 HTMLS 폴더 내에 있는 html 파일을 이용하세요.
* **pc_games_specs_fixed.csv 파일:** PCGamingWiki에서 가지고 오는 게임 정보의 양이 많아 데이터를 모두 가지고 오기까지 시간이 오래 걸립니다. Colab 환경에서 코드 실행시 세션이 끊어질 수 있있으니, 미리 가지고 온 데이터를 이용하세요.(2025.12.07 기준 53222개 데이터 보유, 5시간 36분 소요)
